# 更新日志

## 2026-03-25 - unreleased

### 🖼️ 图片压缩策略调整
- 修复 PNG 压缩时可能出现的图像处理组件错误
- 调整图片压缩策略：非 `jpg/jpeg` 文件统一先转换为 `jpg`，再按 JPG 质量压缩
- 明确透明 PNG 转换为 JPG 时会丢失透明通道，并以白色作为背景底色

### 🧩 稳定性修复
- 修复 QuestPDF 在应用启动阶段过早初始化导致主窗口无法打开的问题
- 将 QuestPDF 初始化延迟到实际执行 PDF 导出时，避免未使用 PDF 功能时启动崩溃

### 📚 文档更新
- 更新 README 中的 `dotnet run` / `dotnet publish` 命令，统一显式指定 `ImageCompressor.csproj`
- 补充 `.slnx` 发布命令的注意事项，避免 `MSB4126` 解决方案配置错误
