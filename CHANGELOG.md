# 更新日志

## 2026-03-20 - v1.2

### 🚀 兼容性与发布
- 项目从 `.NET 10` 迁移到 `.NET Framework 4.8`，提升 Windows 10 兼容性
- 新增 `.NET Framework 4.8` 启动环境检测与异常提示
- 提供 `x86` 和 `x64` 双架构发布版本，适配 32 位和 64 位系统

### 🧩 功能与体验优化
- 优化 PDF 导出流程中的进度弹窗，修复按钮显示不完整问题
- 增强 PDF 导出状态提示，显示当前处理进度和文件名
- 统一 PDF 进度弹窗视觉样式，使其更贴近主界面风格

### 🛠️ 工程调整
- 项目版本号更新为 `1.2.0`
- 新增双架构发布脚本与整理后的分发目录
- 更新文档和启动脚本，统一为 Windows 10 / .NET Framework 4.8 发布方式

## 2025-06-20 - v2.0

### 🎨 UI 重大升级
- **全新Office风格设计**：采用现代化简约风格，类似Microsoft Office界面
- **系统默认窗口**：恢复带标题栏的窗口，移除自定义无边框窗体
- **简洁配色方案**：白色和浅灰色背景，蓝色主题色 (#0078d4)
- **紧凑布局**：优化的间距和边距，去除多余装饰元素
- **标准化控件**：使用系统默认控件样式，提供一致的用户体验

### 🎯 主要改进
- 移除渐变背景和复杂的阴影效果
- 简化拖放区域设计（白色背景+灰色边框）
- 优化字体大小和间距
- 统一按钮和文本样式
- 改进响应式布局

### ✅ 功能保持不变
- 拖放图片压缩功能
- 目标文件大小选择（100KB-5MB+自定义）
- 两种压缩模式
- 压缩结果对比显示
- 自动保存到"压缩后"文件夹

### 🛠️ 技术更新
- 重构 MainWindow.xaml 布局
- 更新 MainWindow.xaml.cs 事件处理
- 优化颜色常量使用
- 重新编译 Release 版本
